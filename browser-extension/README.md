# PrivShield 浏览器插件

这是立项书中“浏览器插件 + 本地服务”承诺的可运行 Manifest V3 原型。插件不会调用第三方云 API，默认只连接本机 `127.0.0.1:8000` 的 PrivShield 后端。

## 安装

1. 先启动后端与前端。
2. Chrome/Edge 打开扩展管理页，开启“开发者模式”。
3. 选择“加载已解压的扩展程序”，指向本目录 `browser-extension/`。
4. 在网页中选中文本，右键选择“用 PrivShield 脱敏选中文本”，或点击工具栏图标。

## 功能

- 读取网页选中文本或当前页可见文字；
- 选择掩码、语义伪名、层级泛化和三级保护强度；
- 调用本地规则、NER、可选 14B 核验流水线；
- 在弹窗内继续编辑并自动保存人工最终稿，也可手动保存、复制或导出 TXT；
- 一键打开完整工作台，继续实体级人工复核、最终稿编辑、审计和导出；
- 设置页可切换到自托管服务器地址。

浏览器插件读取的是网页可见文本，不处理扫描图片，也不会保留网页账号或 Cookie。
